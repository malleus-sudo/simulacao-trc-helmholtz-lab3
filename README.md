# Simulação de feixe de elétrons passando pelo campo magnético de bobinas de Helmholts
## Descrição
Repositório criado para o Experimento 2 - Atividade 3: Movimento de elétrons sob ação de campos magnéticos de Lab 3. 

O objetivo é criar um modelo teórico através de simulação numérica que utilize o mapeamento do campo magnético no interior das bobinas de Helmholtz para simular o comportamento da trajetória dos elétrons que passam por elas utilizazano as equações de movimento e a Lei de Biot-Savart. Depois de feita a simulação teórica, o simulador deve realizar a comparação entre o modelo e os dados das trajetórias obtidos experimentalmente na tela do TRC.
